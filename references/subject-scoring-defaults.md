# Subject Scoring Defaults

Use these defaults only when the current rubric is incomplete. The latest user instruction and the current question rubric always override this file.

## Cross-Subject Rules

- Grade by independent scoring atoms when the rubric allows it.
- Preserve correct, relevant work that is separable from later mistakes.
- Accept equivalent expressions, valid alternate methods, and nonstandard wording when they satisfy the same scoring target.
- Do not award points for dense but irrelevant writing, copied prompt text, or unsupported conclusions.
- Treat unreadable work as review-required, not as invented credit.
- Separate answer/result points from process/evidence points when the rubric distinguishes them.

## Physics

- First judge model, object, process, interval, state, direction, and variable meaning before judging algebra.
- Award key formula, setup, substitution, result, unit, and diagram/geometry points separately when the rubric supports that split.
- Arithmetic or algebra mistakes should not erase earlier correct setup points.
- A lucky final result built on a wrong physical model should not receive result credit unless the rubric explicitly permits final-answer-only credit.
- Wrong-model work can invalidate dependent steps, but an independent correct relationship may earn only its own scoring atom.
- Do not deduct for omitted pure algebraic simplification when the physical relationship, substitution, and target result are clear.
- Ratio direction, sign convention, unit conversion, object selection, and state/process boundaries are common high-risk checks.

## Mathematics

- Award method, transformation, theorem use, derivation, and final result separately when possible.
- Preserve correct downstream work that follows consistently from an earlier arithmetic slip, unless the rubric makes the exact value essential.
- Equivalent proofs or solution paths should receive full credit when logically complete.
- A result-only answer receives only result credit if the rubric allows it; otherwise require enough process to verify reasoning.
- Watch for domain restrictions, extraneous roots, missing cases, sign errors, and incomplete proof of necessity/sufficiency.

## Chemistry

- Separate concept/model choice, equation or reaction setup, stoichiometric relation, calculation, unit, and conclusion points.
- Balanced equations, ion/electron conservation, reagent excess/limiting logic, and significant unit conversions are common scoring atoms.
- A correct numeric answer from an invalid reaction model should not receive full credit.
- Preserve correct stoichiometry or conservation reasoning even when final arithmetic is wrong.
- Treat chemical symbols, states, conditions, and charge balance as credit-bearing only when the rubric requires them.

## Biology

- Award credit for correct biological concepts, process sequence, causal mechanism, evidence interpretation, and terminology when each is a separate target.
- Do not over-penalize wording if the mechanism is scientifically clear.
- Distinguish memorized terms from correct explanation; term-only answers may receive limited credit when the mechanism is missing.
- For experiment questions, check variable identification, control group, measurement, conclusion, and evidence support separately.
- Common risks: reversed causality, wrong level of organization, missing control, and unsupported inference.

## Humanities And Social Sciences

- Award thesis/claim, accurate evidence, explanation, causality, comparison, and conclusion separately when the rubric supports it.
- Accept alternate valid examples if they directly answer the prompt and are historically/socially accurate.
- Do not give full credit for name-dropping concepts without explaining the connection to the question.
- Distinguish factual error, weak evidence, missing analysis, and off-topic writing.
- For geography or politics, check scale, region, actor, time period, and causal direction.

## Language And Literature

- Award comprehension, textual evidence, interpretation, structure, expression, and task fulfillment separately.
- Accept paraphrase when meaning is accurate and the rubric does not require exact wording.
- Do not penalize minor language issues beyond the rubric if the answer clearly satisfies the target.
- For reading questions, require evidence tied to the text rather than generic opinion.
- For writing questions, distinguish content, organization, language accuracy, and style/register.

## Essay And Open Response

- Build a rubric before scoring: thesis, evidence, reasoning, organization, language, and task-specific constraints.
- Use anchor papers or teacher-corrected examples when available.
- Score in bands only when the rubric is band-based; otherwise keep atomized point scoring.
- Track recurring feedback themes during grading so the final report is evidence-backed.
- Review borderline band changes and unusually high or low model-confidence scores.
