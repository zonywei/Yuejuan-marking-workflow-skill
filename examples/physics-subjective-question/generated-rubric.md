# Generated Rubric

## 总分

12 分

## 第 1 小问：电流计算，4 分

评分点：

- 正确写出或使用 `P = UI` 或 `I = P / U`：1 分
- 正确统一单位，将 `120 kW` 转为 `1.2 x 10^5 W` 或等价处理：1 分
- 正确求出发电机端电流 `480 A`：1 分
- 正确求出升压后输电电流 `12 A`：1 分

等价表达：

- `10 kV = 10000 V`、`1.0 x 10^4 V` 均可。
- `120 kW / 250 V = 480 A` 可直接写，不强制写出所有单位换算步骤。

常见扣分：

- 只算出一个电流，最多 2 分。
- 把 `10 kV` 当作 `10 V`，对应电流计算不得该点，但若公式思路正确，可保留公式与单位意识分。
- 电流单位漏写但数值正确，一般不单独扣分，除非导致量纲混乱。

## 第 2 小问：损耗与效率，5 分

评分点：

- 正确使用线路损耗公式 `P_loss = I^2R`：1 分
- 使用升压后的输电电流 `12 A` 计算损耗：1 分
- 正确算出 `P_loss = 2880 W = 2.88 kW`：1 分
- 正确算出用户得到的功率 `117.12 kW` 或等价值：1 分
- 正确算出效率约 `97.6%`：1 分

等价表达：

- 效率写成 `0.976`、`97.6%`、约 `98%`，只要过程清楚均可。
- 如果直接用 `(P - P_loss) / P` 得到效率，也可。

常见扣分：

- 用 `480 A` 计算升压后的线路损耗，损耗分不得，但公式分可得。
- 损耗单位从 W 到 kW 换算错误，视影响扣 1 分。
- 只写效率结论没有损耗过程，最多 2 分。

## 第 3 小问：不升压问题与物理原因，3 分

评分点：

- 正确指出不升压时电流为 `480 A` 或使用该电流分析：1 分
- 正确计算或说明低压输电线路损耗极大，`P_loss = 4.608 x 10^6 W`，远大于输出功率：1 分
- 正确解释高压输电的原因：同功率下提高电压可减小电流，因 `P_loss = I^2R`，电流减小会显著降低线路损耗：1 分

等价表达：

- 可以写“直接低压输电不可行”“线路损耗过大，用户几乎得不到有效功率”。
- 只要明确高电压降低电流、降低 `I^2R` 损耗，即可得解释分。

## 0 分规则

- 答案与本题无关，或只抄题干，不给分。
- 完全空白、无法识别、只写“不会”等，给 0 分并标记 `review_required`。
- 若出现明显非物理结论且没有任何可评分公式或推理，如“电压越高损耗越大所以不用升压”，对应小问不得分。

## 只写最终答案但思路错误的处理

- 如果最终数值正确但推理明显错误，不按满分处理。可给对应数值点，但不给公式或解释点。
- 如果没有过程且答案只包含两个电流值，第 1 小问最多 2 分。
- 如果没有过程只写 `97.6%`，第 2 小问最多 1 分，除非同卷其它位置能看到损耗计算依据。

## 结果错但过程合理的处理

- 公式、单位、物理思路正确，但因算术错误导致最终数值错误，应保留过程分。
- 一个早期数值错误导致后续连带错误时，后续若逻辑一致，可给连带过程分，但最终数值点不得。
- 对低置信度、字迹不清、疑似漏页或疑似空白卷，优先进入人工复核，不自动提交。
