# Physics Subjective Question Example

This is a fictional high-school physics example for demonstrating the workflow. It contains no real student data.

## 题目：远距离输电

某小型水电站向远处用户输电。水电站输出功率为 `120 kW`，发电机端输出电压为 `250 V`。为减少输电损耗，先用理想升压变压器把电压升高到 `10 kV` 后再输电。输电线路的总电阻为 `20 Ω`，忽略变压器损耗。

请回答：

1. 发电机端电流和升压后输电线中的电流分别是多少？
2. 升压输电时，输电线上的功率损失是多少？输电效率约为多少？
3. 如果不升压而直接用 `250 V` 输电，根据计算会出现什么问题？说明远距离输电为什么要采用高电压。

## 分值

总分：12 分

- 第 1 小问：4 分
- 第 2 小问：5 分
- 第 3 小问：3 分
