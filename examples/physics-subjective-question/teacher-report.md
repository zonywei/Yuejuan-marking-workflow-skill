# Teacher-Facing Report

This report is generated from fictional sample answers. It demonstrates the kind of teaching feedback the workflow aims to preserve.

## 阅卷概况

- 题目：远距离输电计算题
- 样例答卷数：3 份
- 满分：12 分
- 样例分数：12、6、0
- 需要人工复核：2 份

## 本题难点

1. `kV` 到 `V` 的单位换算。
2. 区分发电机端电流和升压后的输电线电流。
3. 在线路损耗中使用升压后的输电电流。
4. 理解 `P_loss = I^2R` 中电流对损耗的平方影响。

## 共性错误

- 把 `10 kV` 误写为 `1000 V`，导致输电电流从 `12 A` 错成 `120 A`。
- 用低压侧电流 `480 A` 去计算升压输电的线路损耗。
- 只写“高压危险”而没有解释远距离输电为什么采用高电压。
- 忘记把 `W` 和 `kW` 统一，导致效率计算错误。

## 高分卷特征

高分答卷通常有以下特征：

- 明确写出 `P = UI` 与 `P_loss = I^2R`。
- 分清 `250 V` 端和 `10 kV` 端的电流。
- 损耗、用户端功率和效率的单位一致。
- 对第 3 小问能从“功率一定时电压升高、电流减小、I^2R 损耗降低”这条链路解释。

## 0 分卷分析

样例中的 0 分卷不是因为某一个计算错误，而是缺少可评分证据：

- 第 1 小问无公式和数值。
- 第 2 小问空白。
- 第 3 小问回答偏离题目要求，没有说明高压输电降低线路损耗。

实际阅卷中，疑似空白或低置信度卷应先确认图片完整性和识别质量，再最终确认 0 分。

## 教学建议

1. 在讲解远距离输电时，用同一组数据对比 `250 V` 和 `10 kV` 的电流与损耗。
2. 强调“功率一定”这个前提，否则学生容易只记结论。
3. 单独训练单位换算，尤其是 `kV`、`kW` 与基本单位。
4. 让学生解释公式背后的因果链，而不只是代数计算。

## 阅卷建议

- 对 `10 kV` 换算错误但公式思路正确的答卷，保留过程分。
- 对连带错误要区分“前一步算错导致后续错”和“公式选择错误”。
- 对疑似空白卷、截断图片、无法识别文本，进入复核队列，不直接批量提交。
- 阅卷结束后保留评分标准、结构化评分输出和复核记录，便于教研复盘。
