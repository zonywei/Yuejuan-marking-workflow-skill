## paper-phy-001: 高分卷

1. `P = UI`，水电站端电流 `I1 = 120000 / 250 = 480 A`。升压到 `10000 V` 后，输电线电流 `I2 = 120000 / 10000 = 12 A`。
2. 输电线损耗 `P损 = I2^2 R = 12^2 x 20 = 2880 W = 2.88 kW`。用户端功率约 `120 - 2.88 = 117.12 kW`，效率 `117.12 / 120 = 97.6%`。
3. 如果不升压，电流还是 `480 A`，损耗 `480^2 x 20 = 4608000 W`，比发电功率还大，不能这样远距离送电。高压输电是为了在功率一定时减小电流，从而减小 `I^2R` 损耗。

## paper-phy-002: 部分得分卷

1. `I = P/U`。发电机端 `I = 120000 / 250 = 480 A`。升压后我算成 `120000 / 1000 = 120 A`。
2. 线路损耗 `P = I^2R = 120^2 x 20 = 288000 W`。效率我写成大约 `76%`。
3. 不升压时电流大，导线发热大，所以要升压。升压以后电流小，损耗小。
